# Shop Sales Analysis Project
## Objective 
Analyze shop sales data using SQL and Power BI to identify business insights and trends.

## Tools used 
-Excel
-SQL
-Power BI

## Buisness Questions
-Which month had highest sales?
-Which category generated highest profit?
-Which product performed best?
-Which region generated maximum revenue?

## Project Files 
-Sales_analysis_queries.sql
-Shop_sales_analysis_dashboard.pbix
-Shop_db.csv

## Key Insights 
-Sales increased during festive months
-Technolgy category generated high revenue
-Some product had high sales but low profit
